# **App Name**: SpendWise

## Core Features:

- Login and Registration: Secure user authentication with email/password, Google, and Apple login options.
- Dashboard Overview: Display a summary of the user's financial status, including total balance, spending this week, and recent transactions.
- Expense Tracking: Allow users to input income and expenses with categories, descriptions, dates, and optional receipt uploads.
- Calendar View: Visualize spending patterns on a monthly calendar grid.
- Weekly Spending Summary: Provide users with the current day's transactions and a weekly report to gain insights into spending habits.
- AI Spending Insights: AI spending tool identifies areas where the user can save money.

## Style Guidelines:

- Primary color: Neon green (#39FF14) for actions and highlights.
- Background color: Deep dark green (#0A2613) provides a high-contrast, modern fintech feel.
- Accent color: A lighter shade of green (#6AFF40), analogous to the primary, is used for subtle highlights to maintain focus and consistency.
- Body and headline font: 'Inter', a grotesque-style sans-serif, providing a modern and neutral look.
- Use clean, outlined icons to represent expense categories and actions.
- Emphasize a data-focused dashboard layout with rounded cards and soft shadows.
- Use subtle animations and transitions for a smooth user experience.
